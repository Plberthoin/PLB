# DSPUSRAUT
## Principe
C'est une commande qui permet de savoir le droit qu'a un utilisateur
sur un objet et comment il l'obtient
## Installation
Vous devez compiler le programme et la commande
** le programme doit être compiler avec un profil en *allobj
## Utilisation
DSPUSRAUT OBJET(PLB/VERROU)
          TYPE(*DTAARA)
          USRPRF(FORM01)
Vous recevez un message qui indique comment le profil obtient le droit sur
l'objet
## les objets
* DSPUSRAUT  CLLE
* DSPUSRAUT  CMD
